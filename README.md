# App Finanças 💰

Desafio proposto na criação de telas mobile de um app de finanças.

## 🛠️Tecnologias

- React Native
- React Navigation
- React Native Vector Icons
- Expo

## ▶️Como executar o projeto 

```bash
npx expo start
```
